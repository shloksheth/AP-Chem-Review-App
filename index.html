Unit 7

Molar solubility (s) = moles of a salt that dissolve per liter before equilibrium. Ksp is the equilibrium constant for the dissolving process.

One sentence: Molar solubility is how much of a salt can dissolve in 1 L at equilibrium.

Solids are NEVER in K expressions. Only aqueous ions appear.

CaF₂(s) ⇌ Ca²⁺(aq) + 2F⁻(aq) Ksp = [Ca²⁺][F⁻]² ← solid excluded ICE with s = moles dissolved: [Pb²⁺] = s, [Cl⁻] = 2s Ksp = (s)(2s)² = 4s³ ← (2s)²=4s², NOT 2s²!
⚠ #1 Ksp mistake: (2s)² = 4s², not 2s². The coefficient GETS the exponent applied to it.

Common ion effect: If the solution already has one of the product ions, solubility decreases (Le Châtelier shifts left). Set that ion's initial value ≠ 0 in the ICE table. Ksp itself does NOT change — only temperature changes K.

Unit 8.1

Acid = H⁺ donor. Base = H⁺ acceptor. Every reaction has two conjugate pairs — species differing by exactly one H⁺.

HF + H₂O ⇌ H₃O⁺ + F⁻ Pair 1: HF (acid) / F⁻ (conjugate base) Pair 2: H₂O (base) / H₃O⁺ (conjugate acid)
The +1/−1 rule: Conjugate acid = original + H⁺ (charge ↑1). Conjugate base = original − H⁺ (charge ↓1). NH₃ + H⁺ = NH₄⁺. NH₃ − H⁺ = NH₂⁻.

H₃O⁺ is called "conjugate acid" because it was formed when H₂O accepted a proton in this reaction. In a different reaction it can donate H⁺ on its own. "Conjugate" describes the relationship, not the molecule type.

Unit 8.2–8.3

Water autoionizes: 2H₂O ⇌ H₃O⁺ + OH⁻. Kw = 1.0×10⁻¹⁴ at 25°C — holds in every aqueous solution.

Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴ pH = −log[H⁺] | pOH = −log[OH⁻] pH + pOH = 14

Unit 8.4

Strong acids/bases dissociate completely. [H⁺] = starting concentration. No ICE table needed.

0.050 M HCl → [H⁺] = 0.050 M pH = −log(0.050) = 1.30

7 Strong Acids

  • HCl
  • HBr
  • HI
  • HNO₃
  • H₂SO₄ (1st H)
  • HClO₄
  • HClO₃

Strong Bases

  • LiOH, NaOH, KOH
  • RbOH, CsOH
  • Ca(OH)₂
  • Sr(OH)₂
  • Ba(OH)₂
HF is NOT strong! HF is weak (Ka=7.2×10⁻⁴). Not on the list = weak acid.
Unit 8.5

Ka measures how much a weak acid dissociates. Same structure as Ksp — just an equilibrium constant. Larger Ka = stronger acid.

HA + H₂O ⇌ H₃O⁺ + A⁻ Ka = [H₃O⁺][A⁻] / [HA] ← water excluded ICE → let x = [H⁺] formed: Ka ≈ x²/C → x = √(Ka × C) pH = −log(x) % dissociation = (x/C₀) × 100%
Approximation check: x/C < 5% → valid. If not, use the quadratic. Diluting increases % dissociation (Le Châtelier shifts right).
Unit 8.6

Same ICE logic as Ka, but solve for [OH⁻] first, then convert to pH.

B + H₂O ⇌ BH⁺ + OH⁻ Kb = [BH⁺][OH⁻] / [B] x = √(Kb × C) = [OH⁻] pOH = −log(x) → pH = 14 − pOH
★ Ka × Kb = Kw = 1.0×10⁻¹⁴ (conjugate pairs!) pKa + pKb = 14. Most tested relationship on the AP. Know it cold.
Unit 8.7

Acids that donate more than one H⁺. Each step has its own Ka, always decreasing: Ka1 >> Ka2 >> Ka3.

H₂CO₃ ⇌ H⁺ + HCO₃⁻ Ka1 = 4.3×10⁻⁷ HCO₃⁻ ⇌ H⁺ + CO₃²⁻ Ka2 = 4.7×10⁻¹¹ pH determined by Ka1 only (Ka2 negligible)

Amphoteric species: the intermediate ion (like HCO₃⁻) can act as acid OR base. H₂SO₄ is special: 1st H is strong (complete), 2nd H is weak (Ka2 = 0.012).

Unit 8.8

When salts dissolve, the ion from the weak partner hydrolyzes. The strong partner's ion is a spectator.

Salt frompH

To find pH: Kb = Kw/Ka for basic salts. Ka = Kw/Kb for acidic salts. Then ICE table.

Unit 8.9

A buffer resists pH change. It contains a weak acid + its conjugate base in significant amounts.

How it works: Add H⁺ → A⁻ neutralizes it (A⁻+H⁺→HA). Add OH⁻ → HA neutralizes it (HA+OH⁻→A⁻+H₂O).
Henderson-Hasselbalch: pH = pKa + log([A⁻] / [HA]) When [A⁻]=[HA]: pH = pKa (max capacity) Effective range: pKa ± 1
⚠ Stoichiometry FIRST! If strong acid/base is added to a buffer, calculate new [HA] and [A⁻] after neutralization, THEN use H-H. This is the #1 buffer mistake.

Critical rules and AP exam traps

Before the Quiz

These are the rules that cost points when forgotten. Read once the night before, once the morning of.

Key Rules

Solids & liquids never go in K expressions

Pure solids and H₂O (liquid) are always excluded from Ksp, Ka, Kb, and Kc. That's why the salt isn't in Ksp and water isn't in Ka.

Ka × Kb = Kw = 1.0 × 10⁻¹⁴

Only for a conjugate acid-base pair. Know Ka → instantly know Kb of the conjugate base. pKa + pKb = 14. This is tested constantly.

Temperature is the ONLY thing that changes K

Concentration, pressure, catalysts — these shift Q, not K. Only temperature actually changes the equilibrium constant value.

Stoichiometry before Henderson-Hasselbalch

When strong acid/base is added to a buffer: neutralization stoichiometry first → new [HA] and [A⁻] → then H-H. Skipping stoichiometry = wrong answer.

Diluting a weak acid increases % dissociation

[H⁺] goes down, but the fraction that dissociated goes up. Le Châtelier: adding water shifts equilibrium right.

pH = pKa at maximum buffer capacity

When [HA] = [A⁻], log(1)=0, so pH = pKa. Choose a buffer whose pKa ≈ your target pH. Effective range: pKa ± 1.

Common Exam Traps

1
(2s)² = 4s², not 2s²The coefficient gets the exponent. Always expand the full term. Most common Ksp algebra error.
2
HF is NOT a strong acidMemorize exactly 7 strong acids. Anything else = weak. HF, H₃PO₄, CH₃COOH, HNO₂ are all weak.
3
Conjugate acid vs. base direction+H⁺ = conjugate acid. −H⁺ = conjugate base. NH₃+H⁺=NH₄⁺ (conjugate acid). NH₃−H⁺=NH₂⁻ (conjugate base).
4
Common ion changes solubility, NOT KspMolar solubility drops with a common ion. Ksp is unchanged (only temp changes K).
5
Salt hydrolysis: ask "which partner was weak?"That ion hydrolyzes. Strong partner's ion = spectator always.
6
Polyprotic: use Ka1 for pHKa2 contributes negligibly. Use only Ka1 unless specifically asked about the second dissociation step.
7
Validate the small-x approximationx/C × 100% must be <5%. If over 5%, use the quadratic. AP graders check for this on FRQs.
